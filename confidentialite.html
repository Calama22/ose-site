Hébergement

Le site est hébergé par GitHub Pages (GitHub, Inc., 88 Colin P. Kelly Jr. Street, San Francisco, CA 94107, États-Unis). Le nom de domaine est géré chez OVH.

Données que nous collectons

Le site est un site vitrine. Nous ne collectons aucune donnée personnelle à votre insu :

  • Formulaire de contact : si vous nous écrivez via le formulaire, nous recevons les informations que vous y indiquez (nom, e-mail, message) à la seule fin de vous répondre.
  • Mesure d'audience : avec votre accord uniquement, nous utilisons Microsoft Clarity (voir ci-dessous).

Cookies et mesure d'audience — Microsoft Clarity

Pour comprendre comment le site est utilisé et l'améliorer, nous utilisons Microsoft Clarity, un service fourni par Microsoft Ireland Operations Limited. Clarity enregistre de façon anonyme les pages consultées, les clics, le défilement et le parcours de navigation (« cartes de chaleur » et relectures de session anonymisées).

Clarity n'est activé qu'après votre consentement explicite, via le bandeau affiché à votre première visite. Tant que vous n'avez pas accepté, aucun cookie de mesure n'est déposé et aucune donnée n'est envoyée à Microsoft.
